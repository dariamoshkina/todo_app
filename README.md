# todo_app

## Built with

- Python: 3.7.2
- Flask: 1.0.2
- Vue.js: 2.6.11
- vue-router: 3.2.0
- axios: 0.18.0

## Project setup

### Install backend and serve at localhost:5000
```
pip install -r api/requirements.txt
FLASK_APP=api/app.py flask run
```

### Install front-end
```
npm install
```

### Serve with hot reload at localhost:8080
```
npm run serve
```
